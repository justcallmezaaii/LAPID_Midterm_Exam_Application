
# `LAPID_Midterm_Exam_Application`
CIT18- MASTERY IN WEB

**Objective:**
- You are to develop a simple Laravel application that will display an array of products in an organized manner. This demonstrates your understanding of File Structure and Routing, Controllers, and Models in Laravel.

**Instructions:**
- Set up a new Laravel Project named LASTNAME_Midterm_Exam_Application
- Before the development, you are to select a theme for your products.
	- Gadgets
	- Books
	- Movies
	- Anime
	- Restaurants
- Create a controller named "ProductController" which will home your array of products.
- Define a method in the controller to retrieve and display the array of products based on the selected theme in your ‘blade.php’.
- Push your Laravel project to a GitHub repository under your account. (make it public)
- Repo Name: LASTNAME_Midterm_Exam_Application.
